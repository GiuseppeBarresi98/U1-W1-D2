Come creare un template per i post

di Mario Rossi - pubblicato il 25-10-2020


La prima cosa da fare è suddividere in blocchi logici la pagina,identificando le parti comuni da applicare a tutte le pagine

Le varie aree saranno suddivise in:

  • Contenuti di intestazione del tag <header>
  • Contenuti centrali <main>
  • Contenuti di fine pagina comuni,da inserire nel tag <footer>

Formattazione del codice

Il testo dell'articolo dovrà essere formattato per identificare tutte le parti da selezionare con CSS.Perciò i primi passi saranno:

  1. Progettare il markup assegnando ad ogni specifico testo il tag appropiato
  2. Scrivere HTML nei tag esterni a quelli interni e dell'alto verso il basso
  3. Applicare CSS

L'applicazione di stili CSS è cresciuta negli ultimi anni fino a raggiungere la diffusione pressochè totale.
